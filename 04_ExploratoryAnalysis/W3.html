Prettier dendrograms: a display of hierarchical cluster with coloured leaf labels

myplclust <- function(hclust, lab = hclust$labels, lab.col = rep(1, length(hclust$labels)), 
                      hang = 0.1, ...){
  y = rep(hclust$height, 2)
  x = as.numeric(hclust$merge)
  y = y[which(x < 0)]
  x = x[which(x < 0)]
  x = abs(x)
  y = y[order(x)]
  x = x[order(x)]
  plot(hclust, labels=F, hang=hang,...)
  text(x=x,y=y[hclust$order]-(max(hclust$height)*hang), labels=lab[hclust$order],
       col=lab.col[hclust$order], srt=90, adj=c(1,0.5),xpd=NA,...)
}
myplclust(hClustering, lab=rep(1:3, each=4), lab.col=rep(1:3, each=4))

The picture may be unstable

  • change a few points
  • have different missing values
  • pick different distance
  • change the merging strategy
  • change the scale of points for one variable but it is deterministic should be primarily used for exploration
